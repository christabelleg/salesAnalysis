# Customer Segmentation and Sales Forecasting in Grocery Retail
Engineered customer-level features from 2.4M sales transactions, uncovering 4 segments via K-Means, and training segment-specific classifiers with > 98% next-purchase accuracy to inform churn analysis and targeted marketing. 

[Data Source](https://www.kaggle.com/datasets/andrexibiza/grocery-sales-dataset?select=sales.csv)
